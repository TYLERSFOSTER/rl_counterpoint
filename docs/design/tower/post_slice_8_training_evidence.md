## Limitations

This run is intentionally tiny.

Limitations:

| Limitation | Meaning |
| --- | --- |
| one episode | no musical learning claim |
| one step | only proves plumbing, not long-horizon behavior |
| reward is constant | no TC21M reward quality signal yet |
| terminal_success is false | no cadence success expected from this tiny setup |
| rank 1 only | rank-2 runner is implemented but script parent-policy loading is not |
| checkpoint promotion is budget-based | smarter acceptance remains future work |
